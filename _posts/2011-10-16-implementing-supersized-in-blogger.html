--- layout: post title: Implementing SuperSized in Blogger date: '2011-10-16T09:56:00.009-07:00' author: Prayag Verma tags: - Slider - Code - jQuery modified_time: '2014-07-06T13:47:11.112-07:00' thumbnail: http://2.bp.blogspot.com/-HzSW0M1gBQE/Tpm0Ci3q-NI/AAAAAAAABP4/HqlTpNGPd74/s72-c/Optimized-ss1.jpg blogger_id: tag:blogger.com,1999:blog-2295610366269213074.post-6309265245467964922 blogger_orig_url: http://www.stylifyyourblog.com/2011/10/implementing-supersized-in-blogger.html --- Supersized is a jQuery plugin that automatically resizes (background) images to the size of the browser while maintaining image dimension ratio and runs them as a slideshow in simple words its a Full Screen Slider.

There is no extra whitespace, no scrollbars – the entire browser window is always filled. It is lightweight and is compatible with Firefox, Safari, Opera, IE7, and IE6. It also allows integration with Flickr API.




See Demo



Steps to Add it to Blogger:

1.Login to the Blogger account


2. Now Go to Design > Edit HTML.(For New Blogger Interface Go to Templates > Edit HTML)


3.Now search for the </head> tag and paste the following code just Above/Before it. (How to copy code easily) 
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.6.2/jquery.min.js"></script> 
  
<style>

position:fixed;
left:0;

z-index:-999;

width:100%;
}
#supersized img{

}
