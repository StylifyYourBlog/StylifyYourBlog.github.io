--- layout: post title: Lazy Loading Floating Share Bar Widget for Blogger date: '2012-08-15T12:52:00.003-07:00' author: Prayag Verma tags: - Widgets - Code modified_time: '2014-07-06T14:08:42.332-07:00' thumbnail: http://3.bp.blogspot.com/-_HIHv0SQvV0/UE1pkjNL8II/AAAAAAAAHY4/1doyVPCfGkk/s72-c/sociallazysmall.png blogger_id: tag:blogger.com,1999:blog-2295610366269213074.post-1077842191758005131 blogger_orig_url: http://www.stylifyyourblog.com/2012/08/lazy-loading-floating-share-bar-widget.html --- Presenting the Lazy Loading Floating Social Sharing Bar Widget for Blogger. It provides easy sharing options to your readers without the disadvantage of slow page load times attached with these buttons. The widget uses Socialite to asynchronously load all the scripts and styles related to the buttons as and when the user hovers over the widget. The widget comes in mainly two versions , one with small buttons and the other with large buttons


Demo

Simply select the version that suits you and add it to your blog using the buttons below


Small ButtonsLeft-aligned Right-aligned
On All Pages
On Post Pages
Not on Pages (/p)



Large ButtonsLeft-aligned Right-aligned
On All Pages

Not on Pages (/p)


In case you want to add the code manually
1. Go to Blogger Dashboard , in Layout > Click Add a Gadget > HTML/JavaScript
2. Now paste the following code from below that suits your choice
